About CloudStore

CloudStore is a cloud-based storage service designed to make file management easy, secure, and accessible. Our platform allows users to upload, download, and modify their files effortlessly, leveraging the powerful infrastructure of Amazon Web Services (AWS) to ensure data security and scalability.

Tech Stack

  • Backend: Node.js, Express.js
  • Database: MongoDB (Mongoose)
  • File Storage: AWS S3
  • Frontend: EJS
  • Middleware: Body-parser, Multer (for file uploads)

Getting Started

Prerequisites

  • Node.js and npm installed
  • MongoDB running locally or a MongoDB Atlas connection URI
  • AWS S3 bucket and IAM credentials with proper permissions
